# IGAMING-MASTER-FLOW · Dashboard

Dashboard central de comando do pipeline iGaming.

## Estrutura

```
dashboard/
  index.html   → Dashboard principal (4 painéis)
  config.js    → Credenciais e configurações centrais
  data.js      → Dados dos agentes e memória dos chats
  README.md    → Este arquivo
```

## Painéis

| Painel | Função |
|---|---|
| Saúde dos Agentes | Status dos 23 agentes (L0–L3) |
| Memória dos Chats | Resumo de cada chat ativo |
| Chat Central | Claude API — comando direto |
| Central de Controle | KPIs Supabase + ações n8n |
