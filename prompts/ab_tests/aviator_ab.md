# A/B Test Variants — Aviator

## Test ID: ABT-AVI-001
**Hypothesis**: FOMO-based messaging outperforms benefit-based messaging for Aviator.
**Metric**: Click-through rate (CTR) on outreach messages.
**Duration**: 7 days
**Split**: 50/50

---

## Variant A — Benefit (Control)
```
✈️ Jogue Aviator e multiplique seus ganhos!
💰 Multiplicadores de até 200x disponíveis
🎯 Bônus de R$50 para novos jogadores
👉 Cadastre-se: [LINK]
```

## Variant B — FOMO (Treatment)
```
⚠️ Você está perdendo rodadas de 50x no Aviator!
✈️ Enquanto você lê isso, outros já lucraram
⏰ Não fique de fora — entre agora: [LINK]
```

---

## Test ID: ABT-AVI-002
**Hypothesis**: Social proof copy drives higher sign-up conversion than direct offer.
**Metric**: Sign-up conversion rate
**Duration**: 14 days
**Split**: 50/50

## Variant A — Direct Offer (Control)
```
✈️ Aviator com bônus de 150% no primeiro depósito
💰 Deposite R$20, jogue com R$50
👉 [LINK]
```

## Variant B — Social Proof (Treatment)
```
👥 12.847 jogadores ativos no Aviator agora
📈 Média de multiplicador hoje: 8.4x
✈️ Você é o único fora da festa? → [LINK]
```
